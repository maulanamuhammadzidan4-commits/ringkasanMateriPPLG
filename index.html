Semester 1

Pengertian Pengembangan Perangkat Lunak dan Gim

Pengembangan perangkat lunak dan gim melibatkan proses penciptaan dan pengembangan aplikasi komputer dan permainan video, termasuk desain, pemrograman, pengujian, dan peluncuran.

Keselamatan di Labolatorium Komputer

Di laboratorium komputer, penting untuk mematuhi protokol keselamatan seperti tidak membawa makanan atau minuman, menjaga kebersihan area kerja, dan menggunakan peralatan dengan benar. Maka dikenal istilah 5R: Ringkas, Rapi, Resik, Rawat, dan Rajin atau dalam bahasa jepang 5S: Seiri, Seiton, Seiso, Seiketsu, dan Shitsuke.

Pengertian 5R/5S

  • Ringkas (Seiri)
  • Rapi (Seiton)
  • Resik (Seiso)
  • Rawat (Seiketsu)
  • Rajin (Shitsuke)

Manfaat 5R/5S

  • Mengurangi limbah dan biaya
  • Meningkatkan efisiensi kerja
  • Meningkatkan keselamatan di tempat kerja
  • Meningkatkan kualitas lingkungan

HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat dan menyusun halaman web. HTML menggunakan elemen-elemen (tags) untuk mendefinisikan struktur dan konten dari sebuah halaman web.

Sejarah Singkat HTML

HTML dikembangkan oleh Tim Berners-Lee pada tahun 1990 di CERN. Awalnya, HTML dirancang untuk memudahkan berbagi informasi antar ilmuwan. Seiring waktu, HTML berkembang menjadi standar utama untuk membuat halaman web di internet.

Saat ini, HTML sudah mencapai versi terbaru yaitu HTML5 yang dirilis tahun 2012. Versi ini menambahkan fitur-fitur baru seperti dukungan untuk multimedia dan grafik interaktif.

Kelebihan HTML

  • Mudah dipelajari dan digunakan
  • Berbasis teks, sehingga mudah diedit
  • Dukungan yang luas di berbagai browser
  • Dapat diintegrasikan dengan bahasa pemrograman lain

Kekurangan HTML

  • Batasan dalam hal desain dan interaktivitas
  • Tidak dapat menjalankan logika pemrograman kompleks

Struktur Dasar HTML

Struktur dasar HTML terdiri dari elemen-elemen seperti <html>, <head>, dan <body>. Elemen-elemen ini digunakan untuk menyusun halaman web dengan benar.

CSS

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain dan mengatur tampilan halaman web. CSS memungkinkan pengembang untuk memisahkan konten (HTML) dari presentasi (tampilan), sehingga memudahkan dalam mengelola dan memperbarui desain situs web.

Sejarah Singkat CSS

CSS dikembangkan oleh Håkon Wium Lie pada tahun 1994 saat bekerja di CERN. Tujuannya adalah untuk memberikan cara yang lebih baik dalam mengontrol tampilan halaman web. CSS pertama kali dirilis pada tahun 1996, dan sejak itu telah mengalami beberapa pembaruan, dengan versi terbaru yaitu CSS3 yang dirilis pada tahun 1999.

Kelebihan CSS

  • Meningkatkan fleksibilitas desain
  • Membuat halaman web lebih menarik secara visual
  • Meningkatkan kecepatan pemuatan halaman
  • Mendukung responsif desain untuk berbagai perangkat

Kekurangan CSS

  • Memerlukan pemahaman yang baik tentang selektor dan properti
  • Dapat menjadi rumit untuk proyek besar dengan banyak aturan

Lisensi Perangkat Lunak

Lisensi perangkat lunak adalah dokumen hukum yang menentukan kondisi penggunaan, distribusi, dan modifikasi software. Lisensi ini memberikan hak dan kewajiban kepada pengguna dan pemilik software.

Jenis-jenis Lisensi Perangkat Lunak

  • Perangkat lunak berbayar
  • Freeware
  • Free software
  • Shareware
  • Malware
  • Open source
  • Firmware

Hak Atas Kekayaan Intelektual (HAKI)

Hak atas kekayaan intelektual (Intellectual Property Rights) adalah hak hukum yang melindungi ciptaan intelektual seperti karya sastra, seni, dan temuan teknologi. Hak ini memberikan pemilik ciptaan kontrol terhadap penggunaan dan distribusi karya mereka.

Manfaat Hak Kekayaan Intelektual

  • Melindungi karya cipta dan inovasi
  • Mendorong kreativitas dan inovasi
  • Memberikan insentif ekonomi bagi pencipta
  • Mencegah penyalahgunaan karya orang lain

Jenis-jenis Hak Kekayaan Intelektual

  • Mendorong inovasi dan kreativitas
  • Memberikan insentif ekonomi bagi pencipta
  • Mencegah penyalahgunaan karya orang lain

Ciri-ciri Hak Kekayaan Intelektual

  • Bersifat eksklusif
  • Memiliki nilai ekonomi
  • Dapat dilindungi oleh hukum
  • Bersumber dari kemampuan intelektual

Jenis-jenis Hak Kekayaan Intelektual

  • Hak Cipta
  • Hak Kekayaan Industri (Paten, merk, desain industri, rahasia dagang, dll)
